# 🎙 Emotion Recognition from Speech

## 📌 Project Overview

This project detects human emotions from speech audio using Machine Learning and Deep Learning techniques.

## 🚀 Technologies Used

* Python
* Librosa (Audio Processing)
* TensorFlow / Keras
* Scikit-learn

## 📊 Features

* Extracted MFCC (Mel-Frequency Cepstral Coefficients)
* Used LSTM Deep Learning Model
* Trained on RAVDESS dataset

## 🎯 Emotions Detected

* Happy
* Sad
* Angry
* Neutral
* Fearful
* Disgust
* Surprised

## 📈 Model Performance

Accuracy: ~75%–85%

## 📂 Dataset

RAVDESS Emotional Speech Dataset

## 💡 Future Improvements

* Real-time emotion detection
* Web app using Streamlit
* Improve accuracy using CNN + LSTM
